
\begin{question}

Продать можно только пару ботинок за 600 рублей. По отдельности ботинки
ничего не стоят.

Выберите вектор \((a, b, c)\) выигрышей Ани, Бори и Вовы, лежащий в ядре
этой кооперативной игры.
\begin{answerlist}
  \item (150, 150, 300)
  \item нет верного ответа
  \item (100, 100, 100)
  \item (100, 100, 400)
  \item (200, 200, 200)
  \item (300, 300, 300)
\end{answerlist}
\end{question}
